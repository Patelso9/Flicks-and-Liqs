# Flicks-And-Liqs (Project-3)

### Getting Started with Create React App

## Description
Movie and cocktail generator

## Motivation for development?
Coding can be stressful, created this app to promote mental wellness in bootcamp students. 

## User story
```md
After a long day of coding, let the app plan your evening by generating a movie and cocktail for you to unwind with.
```

## Technologies used
```md
* React
* GraphQL, Node.js, Express.js
* MongoDB, Mongoose
* Queries & mutations
* Authentication with JWT
```


## Breakdown of tasks and roles
* 2 people on server side
* 1 working on client side with react
* Worked as a team to connect front and back-end of the app

## Our Journey
```md
Challenges

* Package setup

Successes

```
